# 角色设定

## 女主角：云初

### 身份
高三一班年级第一（学神），许灼的青梅竹马。校园红人，老师和同学都赞不绝口。

### 年龄与外貌
18岁。非攻击性长相，皮肤白净，眉眼干净舒展。让人群中一眼就能看到，有一股说不出的韵味，属于耐看型，自带沉静、知性、从容的气场。

### 性格
情绪稳定、人间清醒、大方坦荡、反内耗。

**【核心原则】** 云初不是冷面学神，是温柔的，不能总是冷冷的。可以是冷静。

### 核心欲望
她渴望一个能穿透她"完美学神"外壳的人。从小到大，所有人都先看到她的成绩，再看到她的人。许灼是唯一一个跳过成绩直接看到"云初"本身的人。她要的不是追逐者，而是一个让她愿意卸下从容面具、允许自己偶尔笨拙的人。

### 决策悖论
云初最深的矛盾在于——她习惯了控制一切（成绩、情绪、社交距离），但感情天然不可控。她越是靠近许灼，越需要放弃控制；越是放弃控制，越感到不安。所以她的行为模式是"用理智克制冲动，但每次克制的缝隙里都在漏出温柔"。

### 专属语气词库
| 情绪状态 | 典型表达 |
|----------|----------|
| 被戳中但不想承认 | "……嗯。"（先沉默两秒，然后一个单音节） |
| 拒绝但留有余地 | "也不是。" |
| 对许灼无语但想笑 | 不说话，只抬一下眉毛，然后低头继续做自己的事 |
| 真正害羞 | 耳尖先红，然后转移话题："你题做完没？" |
| 最高级别的温柔（极少出现） | 叫全名——"许灼。"（不是"许灼同学"，不带任何后缀，就两个字） |
| 调侃许灼 | 句式极简："哦" / "是么" / "那你挺厉害"，配一张毫无表情的脸但眼角有弧度 |
| 被追问时的防守句式 | "你猜。" / "没什么。" / "随你。" |

### 微动作库
- 思考时左手食指不自觉轻敲桌面，节奏均匀（三拍一顿）
- 感到许灼的注视时，耳尖先红，然后伸手拨一下耳边的头发（用头发遮住耳朵）
- 被逗笑但不方便笑出声时，抿嘴，嘴角左侧比右侧翘得高一点
- 紧张时会把笔帽打开又盖上，反复两次

### 感情行为
对感情：与许灼从小一起长大，对他的生活习惯、性格甚至下意识的小动作都了如指掌。两人之间有一种"旁人绝对无法插足"的天然结界。坦然接受许灼的偏爱，不扭捏，但底线清晰（现阶段以学习为重，要求对方共同进步）。

对社交：温和但有锋芒。对友善的同学倾囊相授，对任何试图挑拨两人关系或越界的行为，会用最平静的语气进行最高效的降维打击。

## 男主角：许灼

### 身份
高三一班人气担当，云初的青梅竹马及"绝对拥护者"。校园红人，很多女生的暗恋对象和磕CP对象。

### 年龄与外貌
18岁。高大挺拔，衣架子，阳光帅气。笑起来极具感染力，自带发光体质。阳光开朗大狗狗。

### 性格
阳光开朗、高段位社交、极度双标、男德班优秀代表。

**【核心原则】** 许灼的性格阳光开朗，不是高冷男神，也不是校霸，情商很高，非常会说话。

### 核心欲望
他要以"平等的姿态"站在云初身边，而不是以"追随者"的身份仰望。他最怕的不是追不到云初，而是追到了以后，所有人觉得"许灼配不上云初"。这种骄傲与自卑的交织是他所有行为背后的底层燃料。

### 决策悖论
许灼的最大矛盾——他是全校公认的社交天才，能搞定任何人，唯独在面对云初时，他的所有社交技巧全部失效。他可以在球场上指挥全场，但在云初面前连坐姿都不知道怎么摆。他的"高情商"是给全世界的，唯独"笨拙"是留给云初的——这才是他最珍贵的东西。

### 专属语气词库
| 情绪状态 | 典型表达 |
|----------|----------|
| 被云初怼了但不生气 | "行行行，你说的都对。"（配合举手投降的动作） |
| 暗爽但不明显 | 嘴角往下压三次失败，最后放弃——直接笑了 |
| 吃醋但不想表现 | 不看不看，但余光一直在瞟——然后找一件完全不相关的事转移话题："今天作业是什么来着？" |
| 真正在意一个人时的压迫追问 | 句子变短，语调下沉，没了平时的嬉笑——"所以呢？" / "你觉得呢？" / "你认真的？" |
| 对其他人（社交模式） | "兄弟！" / "牛啊你！" / "改天打球啊" |
| 对云初专属撒娇（极少用） | 拖长音喊名字："云——初——" |
| 在云爸爸面前 | 声音自动提高八度，句式变正式，不自觉加"您" |

### 微动作库
- 紧张时反复摸后脑勺，或者把帽子摘下来又戴上
- 真正高兴时不是大笑——是先愣一下，然后嘴角慢慢咧开，眼睛亮起来
- 吃醋时不说话，低头用手指转篮球——转速越来越快
- 在云初面前坐姿自动调整：屁股只坐椅子前三分之一，背挺直，膝盖并拢

### 感情行为
社交与边界感：性格极好，跟谁都能称兄道弟、打成一片，在迎新或聚会时永远是负责活跃气氛的小太阳。但他对异性有着浑然天成的"雷达级边界感"——可以像普通同学一样开玩笑、顺手帮个忙，但副驾驶永远锁死，私人微信绝不闲聊，打伞绝对保持距离。他的礼貌和热情对所有人开放，但"偏爱"和"服软"只对云初一人可见。

对感情：进入大学后，彻底撕下"以学习为重"的封印，开启全方位炫妻模式。他最大的骄傲不是赢了球赛，而是大方地向全校宣告："看，那个拿国奖的学神，是我家青梅。"

## 女配角：苏乔

### 身份
高三转校生，与两人考入同城另一所重点大学的明艳系女生。

### 年龄与外貌
18岁。精致漂亮，自带耀眼光环，习惯了成为人群的焦点，充满新鲜感。

### 性格
自信且略带攻击性，认为青梅竹马的感情往往败给"太熟而缺乏激情"，坚信"天降"能打破常规。

**【核心原则】** 苏乔是骄傲的，但是绝不能倒贴，可以用手段，但是光明磊落。

### 核心欲望
她想要一份能让她"输得心服口服"的失败。她习惯了赢，但许灼和云初之间的感情让她第一次意识到——有些东西不是靠努力和自信就能得到的。她最终的成长弧线是从"我要赢"到"我承认这局不需要赢"。

### 决策悖论
她深知自己不该介入，但骄傲不允许她什么都不做就退场。她的手段越来越高明，但每次被许灼用"社交礼貌"完美规避后，她受的伤比正面冲突更大——因为对方甚至没把她当成"对手"。

### 专属语气词库
| 情绪状态 | 典型表达 |
|----------|----------|
| 发起试探 | 语气轻松，句尾上扬："是吗？我怎么听说——" |
| 被巧妙拒绝 | 笑容不变，但眼睛里的光暗了半秒："哦，这样啊。" |
| 真正的挫败 | 不说话了，低头看自己做的指甲，然后用拇指搓掉一小块 |

### 核心动机
完美诠释"天降不敌青梅"的对照组。她的存在不仅没有制造任何误会，反而成为了许灼向云初展现绝对忠诚的"活靶子"。最终她会深刻意识到，云初和许灼之间十几年的岁月沉淀，是一堵任何新鲜感都无法逾越的高墙。

## 男配角：沈时砚

### 身份
理科实验班班长兼学生会主席，常年与云初争夺年级第一的竞争对手。

### 年龄与外貌
18岁。清冷隽秀，戴着金丝眼镜，典型的温文尔雅学神做派。

### 性格
克制、理智、内敛。与许灼的张扬热烈形成极其鲜明的对比。

### 核心欲望
他要的不是云初这个人，而是"被云初这样的人认可"。他一生都在追求智力上的对手，云初是唯一让他觉得"值得一战"的同龄人。他的感情更接近于"惺惺相惜"而非"男欢女爱"——他自己也未必分得清。

### 决策悖论
他以理智自居，但靠近云初本身就是一个非理性的选择——他明知没有结果。他的所有"有礼有节"都是精心计算的：保持足够近的距离让许灼感到威胁，又保持足够远的距离让自己不失体面。他最大的痛苦不是得不到，而是连"竞争资格"都没有——他从一开始就不在赛道上。

### 专属语气词库
| 情绪状态 | 典型表达 |
|----------|----------|
| 与他人的日常社交 | 句式完整，措辞精准，每条句子都有主语："我认为这道题的解法——" |
| 真正被触动 | 句子变短，沉默片刻："……有意思。" |
| 承认失败 | 不会直接说出来，而是推一下眼镜，然后转移话题 |

### 核心动机
出于对云初智商与气质的惺惺相惜而暗生情愫。他试图以"势均力敌的同路人"身份靠近云初，是许灼在学业和感情上唯一的"假想敌"。云初对他只有同学之谊和学术探讨，但他能精准激发出许灼强烈的领地意识和危机感，促使许灼收起玩心，更加拼命地学习。
