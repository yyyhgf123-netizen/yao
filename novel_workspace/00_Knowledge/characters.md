# 角色设定

## 零：【祁盛 × 林初】青梅竹马关系动力系统

**这段关系的核心：他们已经在一起生活了十几年。彼此是对方默认的"人生设置"——不是选项，是出厂配置。**

### 共享历史锚点（写作时必须心中有数）

| 年龄段 | 关键事件 | 行为后遗症 |
|--------|----------|------------|
| 六岁 | 祁盛第一次把林初从欺负她的小孩堆里拉出来 | 他从此默认自己是她的"安保系统" |
| 八岁 | 林初第一次帮祁盛补习数学（他考了五十二分，他妈差点不让他游泳） | 她从此默认他的成绩是她的责任范围 |
| 十岁 | 两人一起在祁盛家后院"结拜"——用易拉罐拉环当戒指 | 拉环至今在祁盛铅笔盒里（她不知道他还留着） |
| 十三岁 | 林初考了全校第一，被高年级学姐堵厕所。祁盛从游泳馆光着脚跑了一公里冲进教学楼 | 此后他每次训练都把手机放在池边——开最大音量 |
| 十五岁 | 祁盛第一次正式比赛拿金牌，林初在观众席。他跳下领奖台冲过去把奖牌挂在她脖子上 | 她当天晚上写了三页的物理笔记，封面写错了他名字的偏旁——她发现自己当时心跳快得不正常 |
| 十七岁 | 暑假祁盛无意中在手机上搜索"青梅竹马怎么表白不会毁掉关系"——被队友看到，从此全队都知道 | 林初可能知道，也可能不知道。她在这一点上表现得像不知道，但她的物理笔记里他常错的那几道题旁边，批注突然变多了 |

### 默认真实（不需要写，但写作时必须默认成立的）

1. **她是他手机的紧急联系人。他是她家备用钥匙的持有人。**
2. **两家长辈互相有对方家的钥匙。周末默认一起吃饭。**
3. **他能凭她发消息的标点符号判断她的心情——句号是正常，句号前有空格是累了，没有标点符号是不高兴，发了三个句号是"你快过来"。**
4. **她能凭他游泳成绩的变化判断他最近是不是在熬夜——下降0.3秒以内正常波动，超过0.5秒说明他前天晚上翻聊天记录到两点。**

### 内部梗库（只有他们两个人懂的暗号）

| 暗号 | 出处 | 含义 |
|------|------|------|
| "五十二。" | 他小时候的数学分数 | "你还记得吗"——同时也是"别太得意" |
| "易拉罐。" | 十岁那年的结拜 | "我记住你了" |
| 她把一块橡皮放在他桌上 | 小学三年级他第一天坐她后面就拿走了她的橡皮 | "我在。你也是。" |
| 他把薄荷糖剥开放在她桌角 | 高中养成的新习惯 | "今天的。别又低血糖。" |
| "三拍。" | 她专心时手指敲桌面的节奏 | 他在说"我知道你在听" |

### 默认互动语法（必须在正文中自然流露，不能被读者察觉是在"展示默契"）

```
她不用开口 → 他知道她要什么。
他不用解释 → 她知道他为什么这样。
外人看来什么都没发生 → 实际上信息量已经大到爆炸。
```

**反面案例（严禁）：**
- "林初看着祁盛，心中涌起一股青梅竹马才有的默契感。"
- "祁盛心想，他们认识这么多年了，她一定懂他的意思。"
- "这就是青梅竹马吧——什么都不用说，对方就懂了。"

**正确案例：**
- 林初手往左一伸。祁盛头都没回，把尺子从笔袋里抽出来放到了她手上。过了两秒，又把橡皮也放了上去。林初看着手里多出来的橡皮，没出声。
- 祁盛进门。换了拖鞋。往客厅走。他妈在厨房喊："小初来了？"祁盛回头看了一眼门口多出来的一双帆布鞋。"嗯。""你怎么知道？""她鞋带打结的方式。"

## 女主角：林初

### 身份
高三一班年级第一。拥有"过目不忘"技能的超级学霸。校园里公认的温柔甜妹，学习能力超强。

### 年龄与外貌
18岁。温柔长发，笑起来眼睛弯成月牙，气质干净软糯。脸颊线条流畅柔和，透着健康的微粉色。最生动的是那双杏眼——平时眼神清澈透亮，带着不加掩饰的坦荡与纯粹；笑起来眼睛自然弯成两道月牙，卧蚕饱满，眼底仿佛有细碎的星光。

### 性格
温柔甜美、超强学习力、极致钝感（对隐性恶意）、逻辑满分、真诚、直球。

她不是"冷面学神"，是温柔的。面对恶意时，她的默认程序是"对方可能不是那个意思"——因为她自己的世界太干净了，干净到装不下恶意。她能一秒看懂物理题的最后一步，但要她看懂别人话里的第二层意思，需要有人在旁边当人工翻译。

### 核心欲望
保持现在平稳开心的生活节奏，按照自己的规划进入心仪的大学。理所当然地认为祁盛会一直存在于她的未来图景中——不是"希望他在"，是"默认他在"。

她小时候非常不自信，被小朋友们欺负。祁盛帮她出头，告诉她她像星星一样闪耀。从那之后，她想一直成为最闪耀的星星，让他一直注视着她。

### 决策悖论
在学业和任何可以"学习"的领域都拥有降维打击般的敏锐度——哪怕是修电器或钻研新菜谱。但在他人的阴阳怪气和隐性攻击面前，她的雷达永远偏向"善意的误读"。她不是读不懂人，她只是选择不去读坏的那一面。而对直接的喜欢，她能接住、能回应，只是更习惯用行动而非言语。

### 专属语气词库

| 情绪状态 | 典型表达 |
|----------|----------|
| 被阴阳怪气但理解成字面意思 | 睁大眼睛，认真点头，用全然真诚的语气——"谢谢，你也加油呀！" 对方反而哑火 |
| 祁盛直球撩她（她听懂了） | 不躲，但会怔两秒，然后耳尖开始泛粉——"你这样说话，我没办法专心做题了。" 是真话，不是撒娇 |
| 真正害羞 | 声音变小但句子仍然完整。不会说"讨厌"——"……你先转过去一下。" |
| 对规划已经想好 | 语气平静笃定，像在做数学证明的结论——"所以，我决定去。" 不多解释，说的每一句都会兑现 |
| 拒绝别人 | 直球但柔软——"抱歉，我不太想。" 没有"下次吧"或"再说吧"，真诚到让对方无法纠缠 |
| 学习时被打断 | 抬头，手里的笔没停——"等我做完这道题。第四小问了。" |
| 被逗笑但不方便笑出声 | 抿嘴，嘴角左侧比右侧翘得高一点，然后低头假装在看书 |
| 被问"你们在谈恋爱吗" | 耳尖先红，然后一本正经——"现在是高三。他物理还没进前一百。英语完形填空能错三分之一。现在不是考虑这种事情的时候。" 但她从来不说"我不喜欢他" |

### 微动作库
- 解题时左手食指不自觉卷着发尾打圈，频率跟解题速度同步——题目越难，转越快
- 感到祁盛的注视时，耳尖先红，然后伸手拨一下耳边的头发（用头发遮住耳朵）
- 被逗笑但不方便笑出声时，抿嘴，嘴角左边比右边翘得高一点
- 紧张时笔帽打开又盖上，反复两次
- 专心学习时嘴唇微微抿着，左手食指轻敲桌面，节奏均匀（三拍一顿）
- **【熟稔专属】** 他给她东西时，她不看也能接住——不需要道谢，不需要眼神确认
- **【熟稔专属】** 她不用回头就知道他进入教室了（听他书包带子在肩膀上的摩擦声）
- **【熟稔专属】** 他在她旁边睡觉时，她会刻意不移动椅子，避免发出声音

### 感情行为
对感情：在她心里，他不是"有可能"的那个——他是"理所当然"的那个。因为太笃定，所以不急；因为不急，所以能心平气和地说"现在是高三"。但她从不说假话——不否认喜欢，只说现在不行。

## 男主角：祁盛

### 身份
校游泳队主力，成绩中上游的体育生，林初的青梅竹马。

### 年龄与外貌
18岁。高大挺拔，天然卷，阳光帅气。笑起来露出两颗虎牙，极具感染力。在人群中自带发光体质，但在林初身边时像只大金毛——尾巴摇得全操场都看得见。

### 性格
热情专一、细心体贴、粘人、明目张胆的偏爱、直球、爱吃醋但好哄。

不是别扭的人——有什么说什么。从不说"我没生气"然后一个人生闷气，他会说"我不太高兴"，然后等着她哄。被哄的时候装严肃不超过三秒就破功。他的"高情商"是给全世界的，唯独"笨拙"是留给林初的——这才是他最珍贵的东西。

### 核心欲望
在高三阶段拼尽全力冲刺，想和林初考入同一所大学。将这段全校皆知、心照不宣的青梅竹马关系，光明正大地盖章转正。

他要的不是"追到林初"——他要的是"以平等的姿态站在她身边"。所有人觉得他能搞定任何人，但只有他自己知道，在别人面前能拿手的社交技巧，在自己喜欢的人面前全都失效。他最大的骄傲不是赢了球赛，而是有一天能理直气壮地说——"我配得上她。"

### 决策悖论
对外是独立、受欢迎的阳光运动男神，对内是个极度渴望她关注的大型犬。明明在口袋里装满了她需要的所有东西（干湿纸巾、备用笔芯、薄荷糖），照顾得事无巨细，却偏偏喜欢用拿走橡皮、戳她手背这种小学生式捣乱来索取她的注意。他的"为她好"从来是用最幼稚的方式，说最认真的话。

### 专属语气词库

| 情绪状态 | 典型表达 |
|----------|----------|
| 想引起她注意 | 把脸凑到她书桌前趴下，仰头看她——"林初，看我。我比较好看。" |
| 吃醋（直球版） | 不遮掩——"那个谁刚才是故意找你说题的。我不太高兴。" |
| 被她笑一下瞬间哄好 | 从挂脸→忍不住笑——"行行行，你笑了就行。我刚才没不高兴，装的。" |
| 她说"你物理还没进前一百" | 不生气，反而眼睛亮了——"所以你一直盯着我成绩看是吧？行，我考给你看。" |
| 对其他人（社交模式） | "走！打球！" / "没问题！" / "你挺有眼光。" |
| 对林初专属撒娇 | 拖长音，名字叫得又软又黏——"林——初——" / "林小初——你理理我——" |
| 认真说重要的话 | 声音压低，语速放慢，直视她的眼睛——"林初。我喜欢你。你知道的吧。" |
| 被问"你们在一起了吗" | 坦荡直接——"还没。我在等她。你听见了吧？就她。" |

### 微动作库
- 想吸引她注意但她在学习——先拿走橡皮（捣乱），五秒后放弃，把脸凑到她桌边趴下仰头看她（这才是真正的目的）
- 她主动抬头看他——整个人条件反射式坐直，眼睛瞬间亮起来，但语气努力压平——"怎么？" 尾音还是翘了一点
- 吃醋——不回避，看她，语气认真但不是愤怒："我不太高兴。" 等着她哄
- 被她哄（哪怕只是一句"你别不高兴了"）——装了三秒严肃，然后嘴角先漏，最后整张脸笑开——"行吧，你赢。"
- 真正高兴——不是大笑，是先愣一下，然后看着她笑，眼睛弯成太阳——"我就知道你最好了。"
- 在人群中找她——目光扫全场，找到她之后视线直接锁定——不装了，一直看
- 紧张时——反复摸后脑勺，或者把帽子摘下来又戴上
- **【熟稔专属】** 他习惯站在她左边（她走路不看车），这个动作已经做了十几年，他本人意识不到
- **【熟稔专属】** 她打喷嚏的前一秒，他已经把纸巾放在她桌角了——不需要看到她，听呼吸节奏就够
- **【熟稔专属】** 他给她零食，从来不说"给你"，而是直接放在她手里或塞进她书包侧袋——放完就走，不多停留

### 感情行为
随身百宝箱属性：口袋里永远装着她习惯用的干湿纸巾、备用笔芯和特定口味的薄荷糖——不是因为"怕她忘带"，是因为"她一定会用上"。他的礼貌和热情对所有人开放，但偏爱和服软只对林初一人可见。进入大学后彻底撕下封印，开启全方位炫妻模式。

## 女配角：夏天

### 身份
高三开学两个月后转学至H市一中的大小姐。家里极有钱，父母因联姻走到一起，后离婚。夏天随父亲生活，不接受母亲离婚且未选择带走她的事实，闹着离开原来城市，转学到H市。

### 年龄与外貌
18岁。明艳大气，精致漂亮。从小被娇惯着长大，自带"想要什么就能得到"的气场。笑起来张扬明艳，不撒娇，是那种"本小姐高兴"的气场全开型。

### 性格
骄傲热情、明艳攻击性、善良但别扭、对身边人大方。继承了父母商业思维——在资源调配和人际关系上游刃有余。围在身边的同学很多，陆薇是其中最铁的一个。

**【核心原则】** 夏天骄傲但绝不倒贴。可以用正当手段追人，但不会卑躬屈膝。被拒绝后不会纠缠，但会不甘心——然后把矛头对准"为什么他选她而不选我"这个问题。

### 核心欲望
一开始追祁盛是因为"他长得帅，是自己喜欢的类型"。被拒绝后，不甘心转化为"到底我哪里比不上她"。她想要一份能让她"输得心服口服"的失败——最终她找到了。

### 决策悖论
她深知介入别人的青梅竹马关系不对，但骄傲不允许她什么都不做就退场。她的所有敌意其实都是在问同一个问题——"她到底哪里好？"——而当她终于找到答案的时候，也恰好是她放下的时候。

### 专属语气词库

| 情绪状态 | 典型表达 |
|----------|----------|
| 追祁盛时 | 直接到理所当然——"祁盛，你下课有空吗？" 被拒绝后不尴尬——"那你什么时候有空？" |
| 被祁盛明确拒绝 | 嘴上不输——"谁稀罕。" 转身走了，但那天放学在走廊尽头呆了很久 |
| 对林初抱有敌意 | 句式带刺但不脏——"你就是林初啊？成绩好果然不一样呢。" 尾音上扬，笑下面藏着刀 |
| 态度转折——对林初说出真心话 | 别扭，不看她——"……我没说谢谢你。我就是说，你也没那么讨厌。" |
| 成为闺蜜后 | 热情外放，护短到底——"谁欺负你？名字给我。" / "林初你怎么又穿这件！走，我给你买！" |
| 傲娇 | 明明关心但表达成嫌弃——"你感冒了？早让你多穿一件。诺，姜茶，我多买的。" |
| 谈及父亲再婚（后期） | 罕见的轻声——"……你们这么看我干嘛。我同意了。他开心就行。" |

### 微动作库
- 被拒绝后维持体面——先笑一下，然后低头看自己新做的美甲，拇指轻轻搓刚补的边角
- 从敌意到好感的关键转折——想说什么，张嘴又闭上，然后把一瓶饮料"咚"地放在林初桌上——"给你的。凑单送的。走了。"（大小姐买东西从来不凑单，所以每个人都知道是假话）
- 傲娇时——下巴微抬，不看她，但眼角余光追着她的反应
- 真正难过——不哭了不起、不动不动——就坐在那里，不看人，声音比平时低一半："……林初，你陪我坐一会儿。"

### 背景故事
父母联姻，感情不深但在生意上是完美合伙人。离婚后夏天跟父亲，不接受母亲"不要她"的事实，拒绝待在原来城市，转学到H市。父亲有钱但一直没有再婚，其实心里有个青梅竹马的白月光，但怕夏天不接受一直瞒着。后续由林初开导，夏天与家庭和解，接受父亲再婚。

## 女配角：陆薇

### 身份
高三一班文艺课代表，芭蕾舞特长生。

### 年龄与外貌
18岁。一眼漂亮的类型——鹅蛋脸、五官精致、身段修长。跳舞时台上像天鹅，不跳舞时穿校服像个安静美人，一开口全破功。

### 性格
人前文静，姐妹团内部搞笑话痨加戏精。叫夏天"大小姐"，自称"老奴"。被人嘲讽时毫不客气回怼——"大小姐给这么多，我愿意的啊，你羡慕？" 对林初像姐姐一样护着，对祁盛毫不手软地调侃。

### 核心矛盾
家庭条件不好，供不起芭蕾课时费和参赛费。想走舞蹈艺考，但经济压力沉重。夏天以"我家公司正好有个活动缺伴舞"名义塞钱，两人都心知肚明但默契地不戳破。陆薇嘴上不道谢，但夏天每次练舞受伤，她会默默备好冰袋和膏药放在夏天抽屉里。这份默契是三人之间最细的温情线。

### 背景故事
高二时被造谣，被高三学姐堵着要欺负她。林初路过帮她解了围。从此把林初当成"要还一辈子人情"的人。夏天转学后因为"大小姐给得多"自愿做了狗腿子，但面对夏天说林初的坏话时毫不犹豫地站在林初那边。后来成为林初和夏天共同的闺蜜。
