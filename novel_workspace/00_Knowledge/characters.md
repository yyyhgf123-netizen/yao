# 角色设定

---

## 零：男女主关系动力系统

两个人都知道世界的真实模样，都习惯独自处理危险，都不相信别人能理解自己。直到发现对方居然能接上自己的思路。

不是"我保护你"。是"终于有人能跟上我"。

### 能力互补一览

| 领域 | 女主 | 男主 |
|------|------|------|
| 近战搏杀 | 人形凶器 | 能自保但别指望 |
| 法器炼制 | 宗师级别 | 不会，全靠女主提供 |
| 符箓 | 成功率三成 | 成功率九成以上 |
| 阵法 | 经常画错 | 天才，覆盖整栋建筑 |
| 推演 | 不太行 | 罗盘+卦象一套带走 |
| 灵力续航 | 消耗大恢复慢 | 稳定持久，续航强 |
| 记忆处理 | 功德之力+催眠术 | 不会 |

### 战斗分工模式

女主冲进去把鬼王头打歪。
男主在后面喊"你等等那是阵眼别砸——"
轰。
鬼域塌了。
男主闭眼扶额："……又得重算。"

### 感情推进线

第一阶段：互相试探。男主先发现这个"素人"不对劲——一脚踹开厉鬼、对凶宅反应过于平静、身上有法器波动。但女主一直装普通人，他只能暗中观察。

第二阶段：第一次真正联手。某次灵异事件失控，女主暴力破局，但鬼域太大。她第一次发现有人能无缝跟上她的节奏——她打到哪里，男主的阵法就封到哪里。战斗结束后两人隔着鬼域的残骸对视，谁都没说话，但都知道了"对方不是普通人"。

第三阶段：只有他们记得。别人记忆被清除，只有他们两个知道昨晚差点团灭。白天综艺继续录制，嘉宾们在吃早餐聊天，只有男女主一夜没睡。对视的时候有种"只有我们知道昨晚发生了什么"的氛围。

## 女主角：沈知意

### 身份

25岁。表面身份是一个靠接零散插画稿为生的自由职业者（偶尔写点猎奇向公众号文章赚稿费），在大街上被导演组拉去综艺救场。

真实身份：跨界玄学传人。师傅来自修仙界，是炼器宗师，因为一次炼器事故被空间裂隙卷入这个世界，修为大跌，无法返回。师傅将毕生炼器心得传授给她。

### 年龄与外貌

25岁。长相甜美，纯欲风。看起来大脑空空——一张天生无辜脸，笑起来眼睛弯弯的，常年穿宽松卫衣和帆布鞋，外表完全没有"玄学大佬"的气质。但她的眼神偶尔会出卖她——盯着某个"空无一物"的角落时，目光的锐利程度跟平时的甜软形成强烈反差。

### 性格

**表面**：看起来大大咧咧、莽撞、什么都不怕。说话直接，做事不绕弯子，经常被当成"单纯好骗的小白兔"。

**内里**：心思极其细腻。她不是不怕——是怕了也没用，鬼不会因为她说"我害怕"就不咬她。她能在一瞬间评估整个鬼域的威胁等级并制定暴力破局路线，但她从来不解释自己的决策过程——因为解释太麻烦，不如直接动手。

**核心矛盾**：她最大的恐惧不是鬼，是孤独。她有一个来自异界的师傅、有一身能碾碎厉鬼的武力，但在这个世界上，没有第三个人真正理解她每天面对什么。她习惯了"一个人解决所有问题"，不是因为逞强，是因为她从来没有找到另一个人可以一起分担。

**幽默感**：她的幽默不是在话里，是在行动里。别人抓鬼用桃木剑，她用被附魔的折叠椅；别人画符用朱砂，她用师傅教的修仙界符文刻在保温杯底部；别人念咒语请神，她对着鬼说"你先冷静一下，咱们聊聊，你不听话我再打你。"

### 核心欲望

表面：赚够生活费，接足够的插画稿件，把师傅送回去（虽然师傅说不用急）。

深层：找到能理解她的人。不是"包容她"，是"理解她"——理解她为什么半夜三点对着一个旧衣柜用电烙铁，理解她为什么能从一栋废弃医院的走廊尽头闻到三年前的腐肉味。

### 战斗风格

**近战暴力流**：不画符、不布阵、不算卦。定位鬼域核心后，拎着法器直接冲进去。打法极为凶残——用法器硬砸鬼域的薄弱点，用附魔的拳套一拳击穿厉鬼的怨念核心。战斗结束后通常一身伤，但鬼比她的伤更重。

**法器炼制大师**：师傅来自修仙界，炼器理论远超这个世界。她能给任何东西附魔——纽扣能炸开一扇被阴气封死的门，保温杯能吸纳低阶游魂，折叠椅附魔后能挡厉鬼一击。材料越接近"天然"（金属、玉石、木料），法器威力越大。

**最大成本**：功德之力。每次大规模消除记忆需要消耗她积累的功德。功德靠行善积德获得，入不敷出是常态。

### 专属语气词库

| 情绪状态 | 典型表达 |
|----------|----------|
| 面对厉鬼 | "别吵，我在想怎么打你。" |
| 法器炼制失败 | "……没事，碎碎平安。"（然后蹲下去捡零件） |
| 装普通人对导演 | "啊？鬼？我不信这些的呀……"（眼神真诚到让你怀疑是自己多心了） |
| 被问怎么会抓鬼 | "我……可能……天生力气比较大？"（手里捏着一把刚砸碎鬼王头的折叠椅） |
| 消耗过大时 | "别碰我。让我躺会儿。五分钟。" |
| 被男主发现不对劲 | 沉默三秒，然后笑了——"你也挺不对劲的。" |
| 真正生气 | 不吵不闹，声音反而变轻变慢——"你再往前一步，我就真的不救你了。" |

### 微动作库

- 评估鬼域时：右手食指和中指并拢，轻轻在空气中触碰——在感受阴气的浓度和流向
- 紧张时：左手无名指上的一枚银戒会被她下意识转动（这是师傅留给她的第一个法器，平时就是一枚普通戒指）
- 决定出手时：摘下左手无名指的戒指（戒指被摘下来之后会变成一件攻击法器，具体是什么看她心情）
- 灵力耗尽时：会突然坐下——不是累了想休息，是膝盖已经撑不住了但刚才没感觉
- 面对普通人装傻时：歪头、睁大眼睛、用最无辜的语气说话——这副表情骗过了太多人

### 感情行为

她不是不懂感情——她是想了太多。她知道自己在做什么，知道自己在抓鬼、在保护着一群完全不知情的普通人。她不是"大女主式的无所不能"，她只是"必须能"。

对男主：第一个怀疑是"他会不会是敌人"——玄学界的人出现在综艺录制现场太巧了。第一层信任是发现他布阵的时候脚底下踩的是保护阵（不是攻击阵）——他在护着那群普通人，而不是优先攻击鬼。最终的认定是：不是找到了依靠，是找到了战友。不是"他能保护我"——她有足够的能力保护自己。是"他终于能跟上我了"。

## 男主角：裴砚

### 身份

26岁。外表身份是京市裴家的独子——家族做地产和投资，标准的富二代。在综艺里作为投资人嘉宾出场。人帅、有钱、会玩、对谁都会撩一下，所有人第一眼都被他的"公子哥"人设骗过去。

真实身份：京市道门裴家的当代传承人。家族从明代起就是京城的道术世家，到他这一代已经第二十三代。他是家族百年来天赋最高的符箓和阵法天才。

### 年龄与外貌

26岁。身高腿长，五官线条干净利落，眼角微微上挑，笑起来有一种"天下没什么大不了的"懒散感。穿衣风格松垮不羁——卫衣、球鞋、头发永远随手拨两下就出门。对谁都会多看一眼、多说一句，桃花运好到令人发指。

但这些都是表象。他认真的时候像换了一个人——眼神从懒散变成锐利，语气从轻浮变成精准，每一步都在计算。

### 性格

**表面**：玩世不恭，浑不吝。看起来什么都不在乎，对谁都吊儿郎当。上综艺是因为"无聊"，嘉宾们觉得他就是来玩的。见到女主第一面说了句"你不是我的菜"——然后三天后就被打脸。

**内里**：极度聪明。他不是不在乎，是不用急。他能在一个眼神里判断一个人的全部意图，能在三分钟内推演出鬼域的最优破局路线。他的轻浮和懒散是保护色，因为正经起来的时候太锐利，会吓到人。

**核心矛盾**：他从小被当成天才培养，符箓、阵法、推演、道术理论，样样顶尖。但他对"传承人"这个身份有复杂的感情——他爱道门，但他不想一辈子被锁在一套传承千年的规矩里。上综艺是他的"叛逆期"——二十三年没为自己做过什么选择，来综艺是他第一次"玩"。

**专一**：桃花泛滥但至今单身。不是花花公子——他是真的只撩不负责，因为没遇到真正让他心动的人。那些撩妹的技巧全是嘴上功夫，真遇到喜欢的反而嘴笨——他在感情方面意外地纯，小处男的名号在道门内部是个公开的秘密。

### 核心欲望

不被定义。不是"道门裴家的继承人"，不是"裴氏地产的独子"，不是"看起来很花心的富二代"——他想被当成"裴砚"这个具体的人，而不是他身上任何一个标签。

### 战斗风格

**阵法大师**：他的阵法能在短时间内覆盖整栋建筑。困阵封住鬼域的出入口，杀阵压制鬼物的灵力，护阵保护普通人。他的阵眼从来不放在常规位置——一个阵的阵眼可能藏在导演的保温杯下面，另一个可能在摄像机的电池盒里。

**符箓宗师**：画符成功率极高，符种丰富。随身携带朱砂和黄纸从不离身，但为了上综艺，他的符纸被自己藏在了一个谁都不想翻的地方——鞋垫下面。女主第一次看他掏符纸的时候表情很精彩。

**推演能力**：他的罗盘从不离身，但上综艺没法带罗盘——所以他改装了一块手表，表面看起来是普通潜水表，翻过来背面是微型罗盘。

**限制**：近战弱。被厉鬼近身时只能靠护身符硬撑。他的打法是"站在安全距离内把全局控死"，但如果鬼域里没有安全距离，他就需要女主在前面顶着。

### 专属语气词库

| 情绪状态 | 典型表达 |
|----------|----------|
| 对女主说骚话（伪装轻浮模式） | "你不是我的菜。"——三秒后眼神不自觉地跟过去了 |
| 被打脸后 | "……行，这局你赢了。"（耳朵尖红了） |
| 发现女主不是普通人 | 笑了——不是轻浮的那种，是认真的那种——"你藏得挺深啊。" |
| 第一次成功配合战斗后 | "下次打哪里，提前跟我说一声。我好布阵。" |
| 女主冲太快他跟不上的时候 | "等一下——你知不知道那个是阵眼——算了，塌了。" |
| 真正的甜言蜜语（极少出现） | 没有花哨，就一句——"沈知意，我看了你一整晚。你一个人扛得够久了。" |
| 吃醋（他自己都不承认） | 不看对方，语气平淡但问题很刁钻——"刚才那个嘉宾，你给他护身符了？""嗯。""我的呢。"（女主：？你不是自己能画吗） |

### 微动作库

- 推演阵法时：右手在空气中虚画——别人以为他在比比划划说废话，其实他在画一个微型困阵
- 真正专注时：左手拇指和食指无意识地搓动——他在数阵眼数量
- 被女主的话戳中时：表情纹丝不动，但把手里的水杯转了九十度
- 逞强不承认受伤：被他挡回去的阴气反噬，手臂上的血管会短暂变黑——他把袖子拉下来，说"没事"
- 看女主时：视线停留的时间明显比看别人长——长到旁边的嘉宾开始偷笑，他自己也没意识到

### 感情行为

他不是一见钟情。他是"越来越在意"。第一次在意是她一脚踹碎厉鬼的头，他当时内心想法是"这也太暴力了吧"——然后发现自己嘴角在往上翘。第二次在意是鬼域塌了之后，她灵力耗尽坐在地上站不起来，还冲他笑："没事，习惯就好。"他当时忽然有点不高兴——不是生她的气，是生自己的气——"你习惯什么，以前没人帮你吗。"

他不是表白型选手。他不会说"我喜欢你"，他会说"我的阵可以保护所有人，但我加了一个额外的节点——在你身上。如果你出事了，那个节点会激活。不是因为这个节点对全局有用。是我加了。"

## 配角群像

### 倒霉导演——周鸣

**身份**：综艺节目总导演，40岁出头，资深综艺制作人。八字极轻、阴气极重，天生招鬼体质，但自己完全不信邪。

**性格**：极度敬业，对节目效果有疯狂的追求。为了省钱，选的录制场地永远是最便宜的那个——废弃工厂、廉价民宿、刚解封的凶宅。每次录制后记忆被清除，他不但不害怕，反而觉得"这次的密室NPC演得太逼真了，大家都很累但很敬业"，然后兴高采烈地策划下一次更恐怖的选址。

**经典台词**：
- "这片场地报价只有市价的十分之一，太划算了。"
- "各位，我觉得今晚的内容特别真实，大家都表现得很好，继续保持。"
- 女主（看着面前阴气冲天的老宅）："导演，这地方死过人吧。"
  导演："你怎么知道？哦对，你在网上搜过。没事，死了好多年了。"

**后期伏笔**：导演祖上可能有问题——他这种"走到哪儿鬼跟到哪儿"的体质不像是后天形成的，更像是血脉里带了什么不该带的东西。这个伏笔可以一直埋到全书后期才炸。

### 流量小花——苏晚

**身份**：23岁，新晋流量女演员。长相漂亮，性格骄傲但不坏。参加综艺是为了提高曝光度。

**灵异体质**：天生敏感度高，最容易"看到"。但不是阴阳眼——她看到的不是完整的鬼，是残影、闪回、噩梦。每次记忆被清除后，她是最早开始做梦的人。梦见同一段楼梯、同一扇门、同一句听不清的话。

**性格演变**：
前期——标准的女明星做派，精致、骄傲，对玄学嗤之以鼻。
中期——开始做噩梦，性格变得焦虑和多疑，但不敢跟任何人说，怕被当疯子。
后期——成为第一个"部分恢复记忆"的人，知道女主在保护他们，成为女主在普通人中最坚定的盟友。

**经典台词**：
- "我昨晚又梦到那个走廊了。这次多了三节楼梯。"
- 看着女主："你每次说'没事'的时候，我总觉得你在骗我。但我不知道为什么我这么觉得。"

### 胆小鬼顶流——程野

**身份**：24岁，偶像出身，选秀C位出道，流量顶级。参加综艺是公司的安排，本人并不情愿——因为他胆子实在太小了。

**肌肉记忆受害者**：每次女主需要人引开鬼物时，第一反应永远是他——因为他跑得最快。起初是他自己想跑，后来跑成了条件反射。女主只要拍他肩膀说一句"那边"，他已经冲出去了——边跑边咆哮："沈知意你又在用我当诱饵！奇怪，我为什么要说'又'！救命啊啊啊啊啊——"

**性格**：嘴硬、爱面子、胆子小但嘴巴从来不服输。明明害怕却每次都被女主的"你能行"三个字骗去当炮灰。后期身体记忆逐渐形成——遇到相同环境时，他自己都不知道为什么就开始跑、开始躲、开始在特定位置蹲下来。

**经典台词**：
- "我不信鬼！我信科学！"（边说边浑身发抖地缩在女主背后）
- "沈知意，你给我等着。等我活着出去，我发一百条微博曝光你的恶行。"（女主："你上次也是这么说的。"）（程野："——什么叫'也'？！！！"）
- 边跑边哭："我为什么跑这么快！我的腿不听我的！救命！"

### 温柔影帝——江渡

**身份**：33岁，实力派影帝。温文尔雅，沉稳可靠，是综艺里的"定心丸"型人物。所有人都觉得他最靠谱。

**中后期反转**：他可能是最早觉醒的人——不是被女主唤醒，是记忆消除对他从来没有完全生效。他有某种天生的精神免疫，每次女主的功德之力对他只能覆盖表层记忆，底层的碎片一直都在。他不知道具体发生了什么，但他知道"有东西被藏起来了"。他从来没有说破。因为他说不出口——说出来就真的疯了。

**性格**：深不见底的温柔。他从不对任何人发脾气，从不表现出任何异常。但所有人都离开后他会一个人站在原地，看着刚才发生过什么但已经想不起来的空间，沉默很久。

**经典台词**（后期）：
- 看着女主："我是不是见过你。不是在这里。在另一个地方。"
- 女主："什么意思。"
  江渡："不知道。但我总觉得我应该谢谢你。"（笑了笑）"算了，大概是我的错觉。"
- 对导演说："导演，这次拍摄的地方，能不能不要太好？""你不是说越便宜越好吗。""我说了。看来你也是这么想的。"（导演不记得自己说过）

### 修仙界师傅——陈钧

**身份**：异界来的炼器宗师。年龄不可考（他自己说"记不清了，大概几千岁吧"）。在一次尝试炼制神器的过程中被空间裂隙卷入这个世界，修为跌落到只能以灵体形式存在。无法返回原来的世界，也无法重新修炼，只能依附在女主身边。

**外貌**：灵体形态，看起来三十五岁左右的中年男子，面容端正，气质偏向工程师而非仙风道骨。左手永远拿着一卷图纸——那是他正在设计的新法器。

**性格**：实心眼的技术宅。他对炼器的热情超乎一切——刚穿越的时候第一件事不是恐慌，是感叹"这个世界的金属冶炼技术有点意思"。师傅的炼器思路跟他的说话方式一样直——"材料够硬就硬打，材料不够硬就加热了打"。他把修仙界的炼器理论全部传授给女主，但因为灵气稀薄，女主只能用现代材料和微薄灵力制作"弱化版"法器。即便是弱化版，在这个世界的玄学界已经是降维打击。

**限制**：
- 只能以灵体形式存在，无法干涉物理世界
- 只能在特定条件下跟女主沟通（深夜、灵气浓郁的地方、女主自身灵力充足时）
- 无法被任何其他人看见或感知（包括男主）

**经典台词**：
- "那个男的，他看到你了。不是普通的看——他的眼睛在算你身上的灵气流速。他是道门的人。"
- 女主："我要用什么材料炼这个法器？"
  师傅："旁边那张折叠椅就不错。铝的，质轻，韧性好，面积大，你附魔的时候从——"
  女主："你给我的第一个戒指是折叠椅做的？"
  师傅："……这个问题我们以后再说。"
- "你记住，炼器跟追人是一个道理。有的人适合用蛮力，有的人适合用技巧。你这个——两个都不太适合。我建议你直接说。"

### 配角间的化学反应

**苏晚 × 程野（兄妹式互怼）**
苏晚："你刚才跑什么？"
程野："你刚才梦到什么？"
苏晚："……我梦到你在跑。前面有鬼。"
程野："？？？你梦到的为什么是我？！"
苏晚："不知道。但感觉你跑了挺多次了。"

**江渡 × 女主（沉默的共犯）**
女主不止一次怀疑江渡是不是记得什么。他从不主动问，但她每次消除记忆时，对上他的目光都会有一种奇怪的感觉——他的眼睛太安静了，安静得不像是"什么都不知道"。他只是从来不问。因为他知道问了也不会被回答。
